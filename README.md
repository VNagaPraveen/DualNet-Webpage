# DualNet Webpage

A modern, professional website for **DualNet** - a logistics network planning solution that brings all your pallet networks into one place for smarter, automated planning.

## 🎯 Overview

DualNet simplifies pallet network management with:
- **80% less time** spent on planning
- **Zero double keying** of data
- **90% fewer calls** chasing updates
- **Live in 24 hours** with no setup required

## ✨ Features

### Website Sections
1. **Navigation Bar** - Sticky header with smooth scrolling
2. **Hero Section** - Eye-catching landing with key statistics
3. **How It Works** - 3-step process explanation
4. **Features Comparison** - DualNet vs traditional alternatives
5. **Benefits Showcase** - Hexagon visualizations and customer testimonial
6. **Call-to-Action** - Free demo booking section
7. **Footer** - Company links and information

### Design Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Modern Aesthetics** - Professional yellow and dark gray color scheme
- **Smooth Animations** - Scroll effects and hover interactions
- **Interactive Elements** - Smooth scrolling navigation and demo button
- **Professional Layout** - Clean typography and spacing

## 🛠️ Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with CSS Grid and Flexbox
- **JavaScript** - Smooth scrolling, animations, and interactivity

## 🔧 Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-yellow: #FFD700;
    --dark-gray: #2a2a2a;
    /* ... other variables */
}
```

### Update Content
Edit text in `index.html`:
- Hero headline and description
- Feature benefits
- Testimonials
- Footer links

### Modify Layout
Adjust grid layouts and spacing in `style.css` for different designs.
